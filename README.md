## LOCTRAC SOFTWARE INSTALLATION

COMPATIBLES DISTROS:
- Linux (Kali Linux)

``
git clone https://github.com/PENTAGONE-GROUP/loctrac.git
``

``
cd loctrac | chmod +x main.py
``

``
pip3 install -r requirements.txt | python3 main.py
``
